Styling and Navigation Bar


The navigation bar has an elegant design with a striking color palette with active links in #353535 and hover effects styled with #d2e12c. All the fonts are set in Trebuchet MS ensuring elegance and readability throughout the webpage. The system is built with flexbox used for the navigation, which helps with a responsive layout that centers everything and makes it easy to navigate the website, increasing user-friendly design and integrating a style that elevates the visual aesthetic.

HomePage


As Student 2, I designed a homepage with a logo, navigation bar, footer with editor page links, and four content-related images. Having an external CSS file allowed to keep the same style throughout the site. I shared this file with the other members of my team and worked with them to help guide them through the process, to ensure a smooth integration and troubleshoot any issues which they may have had.

Feedback


Feedback page layout provides a formal structured interface that prioritises user interaction. It consists of textboxes for name, email, and phone number, and radio buttons to rate the site navigation and service satisfaction. The layout was created with a minimal and clean style using Trebuchet MS font to offer a user-friendly and clarity experience. In addition, JavaScript confirmation was added for each field to ensure no required details were missed.

Curriculum and Instruction


The HTML page is themed on Curriculum and Instruction, with the document divided into four key areas. The document attends to access to education and ensuring that all children receive equal opportunities regardless of economic disadvantages. It highlights relevant curricula focusing on subjects like mathematics and science, which equips students to meet the challenges of the future. The document calls attention to the significance of the teacher in influencing the learning outcomes of the students.
